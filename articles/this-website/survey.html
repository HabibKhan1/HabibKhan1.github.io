The Website's "Hidden" Survey

My website (run on GitHub pages) has a not-so-visible survey page. This was a fun project to make. Basically, it generates a limited-time code for the user to enter on this page (like XXXXX-XXXXX). The underlying program to generate this code and to push the changes (with Git) is on this repository. It is mostly written in C, while also using Batch and PowerShell (Windows only!)

The user enters their e-mail, checks their e-mail, follows the link, and enters their code.

The program will get the e-mail, generate the code, push the changes to GitHub, and send the e-mail with the code.
